# Project Manifesto

## Purpose

Project Genesis exists to create a playable fantasy online RPG foundation with the soul of old-school MMORPGs and the production discipline of a small modern indie team.

The project should feel like the beginning of a living world: a village, dangerous roads, weak enemies near safety, stronger enemies deeper in the wild, meaningful loot, visible character growth, and a sense that the player is becoming part of something larger.

## What This Game Should Be

The game should be:

- small at first, but complete;
- readable to the player;
- expandable by design;
- built around clear systems instead of one-off hacks;
- atmospheric without needing huge content volume;
- friendly to solo or tiny-team development.

The first successful version is not a massive world. It is one polished loop:

1. Create a character.
2. Enter the starter village.
3. Talk to an NPC.
4. Accept a quest.
5. Fight enemies.
6. Gain experience.
7. Receive loot.
8. Equip an item.
9. Return for a reward.
10. Feel stronger.

## What This Game Should Not Be

The game should not become an uncontrolled clone of a large commercial MMORPG.

Avoid:

- starting with huge maps;
- building many races before one race feels good;
- building many classes before one class has complete combat;
- adding multiplayer before the local gameplay loop is stable;
- adding crafting, auction, guilds, raids, and mounts before the first core loop works;
- making systems depend on scene-specific objects;
- writing code that only works for one demo case.

## Decision Rule

When choosing between a bigger feature and a more finished core loop, choose the core loop.

When choosing between visual spectacle and reliable architecture, choose architecture.

When choosing between perfect future flexibility and a working prototype, choose a simple design that can be refactored cleanly.

## Project Identity

Project Genesis is a grounded fantasy RPG prototype with long-term MMO ambitions.

Its strength should come from:

- clean progression;
- understandable combat;
- atmospheric worldbuilding;
- modular systems;
- disciplined scope.
