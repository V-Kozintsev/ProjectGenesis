# Items, Inventory, And Equipment

## Item Types

Initial item types:

- weapon;
- armor;
- consumable;
- quest item;
- material later.

## Equipment Slots

Start with:

- main hand;
- chest;
- legs optional;
- boots optional.

Add later:

- off hand;
- helmet;
- gloves;
- rings;
- amulet.

## Item Rarity

Initial rarity:

- Common;
- Uncommon;
- Rare.

Later:

- Epic;
- Legendary.

## Loot Philosophy

Early loot should be sparse but meaningful.

The player should understand:

- why an item is better;
- whether it fits the class;
- what changed after equipping it.

## Inventory Rules

Prototype inventory can be slot-based.

Needed operations:

- add item;
- remove item;
- move item;
- equip item;
- unequip item;
- check if inventory has space;
- serialize inventory for saving.
