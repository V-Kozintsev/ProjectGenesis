# First Message To Codex

Use this when starting a fresh Codex task for this project:

```text

Сначала:
1. кратко перескажи, что за проект мы делаем;
2. проверь текущее состояние репозитория;
3. предложи план первого маленького шага;
4. укажи, какие файлы планируешь создать или изменить;
5. дождись подтверждения, если задача слишком широкая.

Главный принцип: делаем маленькую, но рабочую RPG-основу. Не строим огромную MMO сразу.
```
